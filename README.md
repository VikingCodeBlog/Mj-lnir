# Install proyect
## Clone
```
git clone https://github.com/VikingCodeBlog/Mj-lnir.git
```
## Install vsce
npm install -g vsce

# Publish your own version
You need an Azure DevOps account.

```
vsce package
```
```
vsce publish
```
